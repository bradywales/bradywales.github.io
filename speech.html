Speech Recognition

Pytorch Machine Learning Model, 86% Validation Accuracy

Skills: Neural Networks, Audio Processing

Machine Learning Model that recognizes speech audio and converts to text.

  • Built using fully connected layers, dropout, and batch normalization.
  • Audio is converted to machine readable format by transforming the data into a spectrogram.
  • Data on the spectrogram is then read by the model to classify the sound produced at each timestep.
  • Experimented with different hyperparameters to achieve the best result.
