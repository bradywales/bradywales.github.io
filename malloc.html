Malloc

Written in C, > 1000 lines of code.

Skills: Low-Level C, Heap allocation, Memory Management

Implementation of C's dynamic memory allocation functions, including malloc, free, realloc, and calloc.

  • malloc (int size):

    Allocates a block of memory of a given size, and returns a pointer to the start of the block.

    If there is no space left in the heap, requests more memory and extends the internal heap.

  • free (void *ptr):

    Frees a block of memory that was previously allocated.

    The newly freed block can now be allocated in future malloc calls.

  • realloc (void *ptr, int size):

    Resizes an allocated block to requested size.

  • calloc (int n, int size):

    Allocates an array of size n, with each element being the same size.
