Facial Recognition + Verification

Pytorch Machine Learning Model, 90% Validation Accuracy

Skills: Convolutional Neural Networks, Image Transformations and Augmentations

Machine Learning Model that recognizes and classifies faces. Dataset consisted of 7000 different faces to classify.

  • Built upon the ConvNext architecture.
  • Model consists of over 25 million trainable parameters.
  • Extracts features from a given image using a series of convolutional layers, then classifies the image using a fully connected linear layer.
  • Experimented with many different variations of hyperparameters to find the best combination.
