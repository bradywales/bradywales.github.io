Seam Carving

Written in Standard ML, < 100 lines of code

Skills: Dynamic Programming, Functional Programming

Seam Carving is a technique for resizing images without distorting the content of the image.

  • This is done by removing seams of pixels from the image that are least important to the content of the image.
  • Seams are continuous paths across the image.
  • Linear time complexity by running in parallel: O(mn)
  • Resizes image based on size parameter given. Size = 100 means the 100 least important seams will be removed.
