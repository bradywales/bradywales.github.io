Zomb112

Written in Python, 1000+ lines of code, semester long term project for the course 15-112

Skills: 3D Graphics, Large Scale Application Development, Pathfinding

A 3D wave based survival game written in Python.

  • Without any existing 3D graphics library, I wrote a custom raycasting function to create the 3D environment.
  • Uses a pathfinding algorithm to allow the zombies to navigate the map.
  • The player earns points for every wave they survive, which they can use to buy upgrades.
